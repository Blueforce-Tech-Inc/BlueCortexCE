2. **批量创建线性扩展**：批量越大，单条平均时间越短（从2.76ms降至1.28ms），说明批量操作有良好的并发处理
3. **并发无退化**：20并发与5并发相比，单请求平均时间反而更低，说明服务并发能力充足
4. **内存使用合理**：RSS 457MB，在Java服务中属于正常范围
5. **搜索性能稳定**：不同复杂度查询时间差异不大（19-24ms）
