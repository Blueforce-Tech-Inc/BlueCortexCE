- Initial release of Cortex Community Edition
- Memory-enhanced agent system with persistent context
- PostgreSQL + pgvector support for vector storage

- WebUI integration
- SSE (Server-Sent Events) support for streaming responses
- Thin Proxy + Fat Server architecture
